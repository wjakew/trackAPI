Map of the service


Maintenance

  • /health - check status of the main components
  • /token-check - check if your personal token is correct
  • /user - get user data by given id (correct app token needed)
  • /login - for logging into app
  • /register - for registering new user to the app
  • /session-validation - for checking if session is still valid
  • /password-reset - for resetting user password
  • /password-check - for checking user password
  • /password-reset - for resetting user password ( for user )

    • Project
  • /project-set - for adding new project to database
  • /project-update - for updating data of project
  • /project-remove - for removing project
  • /project-viewer - for viewing project glances
  • /project-get - for getting project data

    • Task
  • /task-set - for adding task to database
  • /task-update - for updating task to database
  • /task-remove - for removing task from database
  • /task-viewer - for viewing task glances
  • /task-get - for getting task data
  • /task-history - for getting task history
  • /task-archive - for viewint done tasks
  • /task-done - for setting task done
  • /task-open - for setting task to open again

    • Issue
  • /issue-set - for adding issue to database
  • /issue-viewer - for showing issue glances from database
  • /issue-get - for getting issue by issue_id
  • /issue-remove - for removing issue
  • /issue-history - for loading issue history data
  • /issue-archive - for showing archived issues
  • /issue-open - for opening issues again

    • Board
  • /board-set
  • /board-get
  • /board-viewer
  • /boardmanager
  • /boardmanager-remove
