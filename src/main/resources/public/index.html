API SERVER

Map of the service


Maintenance

  • /health - check status of the main components
  • /token-check - check if your personal token is correct
  • /user-configuration - for loading user configuration to client app
  • /user - get user data by given id (correct app token needed)
  • /login - for logging into app
  • /register - for registering new user to the app
  • /session-validation - for checking if session is still valid
  • /password-reset - for resetting user password
  • /password-check - for checking user password
  • /password-reset - for resetting user password ( for user )
  • /user-email-set - for setting email for logged user

    • Project
  • /project-set - for adding new project to database
  • /project-update - for updating data of project
  • /project-remove - for removing project
  • /project-viewer - for viewing project glances
  • /project-viewer-member - for viewing projects with membership
  • /project-get - for getting project data
  • /project-members-add - for adding members to the project
  • /project-members-remove - for removing members from project

    • Task
  • /task-set - for adding task to database
  • /task-update - for updating task to database
  • /task-remove - for removing task from database
  • /task-viewer - for viewing task glances
  • /task-get - for getting task data
  • /task-history - for getting task history
  • /task-archive - for viewint done tasks
  • /task-done - for setting task done
  • /task-open - for setting task to open again
  • /task-comment-list - for listing comments to tasks
  • /task-comment-add - for adding comments to tasks

    • Issue
  • /issue-set - for adding issue to database
  • /issue-viewer - for showing issue glances from database
  • /issue-get - for getting issue by issue_id
  • /issue-remove - for removing issue
  • /issue-history - for loading issue history data
  • /issue-archive - for showing archived issues
  • /issue-open - for opening issues again

    • Board
  • /board-set - for setting new board to database
  • /board-remove - for removing board
  • /board-get - for getting board data
  • /board-viewer - for viewing board content
  • /board-viewer-element - for viewing board element
  • /boardmanager - for adding element to board
  • /boardmanager-remove - for removing element from board

    • Share
  • /shared-viewer-myshares - for showing shared project by user
  • /shared-viewer-sharedtome - for showing shared to user projects
  • /shared-set - for setting project to share
  • /shared-remove - for removing share of project

    • Snippet
  • /snippet-set - for loading user snippets
  • /snippet-remove - for removing user snippet
  • /snippet-viewer - for loading snippet view
  • /snippet-get - for getting information about snippet

    • ToDo
  • /todo-add - for adding todos object to database
  • /todo-list - for getting list of todos
  • /todo-state - for setting state of the todo
  • /todo-remove - for removing todo object
  • /todo-get - for getting todo object

    • Room
  • /room-create - for creating room
  • /room-remove - for removing room
  • /room-data - for getting room data
  • /room-addmember - for adding room member
  • /room-removemember - for removing room member
  • /room-invite - for adding to room with invites
  • /room-message - for getting single message
  • /room-messages - for getting list of messages
  • /room-message-send - for sending room messages
